Pequod Engine

A low-level engine for games that require more control over system resources such as in the simulation genre

Roadmap

A game engine is only as good as the games it can produce and so I've taken on the 20 games challenge (though I'll probably only do around 15 of them) which will allow me to only implement features as I need them, gradually build the project out and also produce demos I can show to people! I have the details here!

Contributing

The main point for obtaining the source code is at git.reocelot.com/pequod-engine The code is also mirrored on github.com/siddhantmadhur/pequod-engine

For contributing to this project, you will have to know the basics of how Gerrit works. I also recommend using JJ-VCS instead of the git cli!
