\hypertarget{md_README_autotoc_md3}{}\doxysection{\texorpdfstring{Roadmap}{Roadmap}}\label{md_README_autotoc_md3}
A game engine is only as good as the games it can produce and so I\textquotesingle{}ve taken on the 20 games challenge (though I\textquotesingle{}ll probably only do around 15 of them) which will allow me to only implement features as I need them, gradually build the project out and also produce demos I can show to people!
