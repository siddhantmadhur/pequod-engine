# Pequod Engine

A low-level engine for games that require more control over system resources such as in the simulation genre

![Demo of Breakout running on the Pequod Engine](https://static.reocelot.com/gif/breakout_demo.gif)

## Roadmap

A game engine is only as good as the games it can produce and so I've taken on the 20 games challenge (though I'll probably only do around 15 of them)
which will allow me to only implement features as I need them, gradually build the project out and also produce demos I can show to people!
I have the details [here](/GOALS.md)!

### Contributing

The main point for obtaining the source code is at [git.reocelot.com/pequod-engine](https://git.reocelot.com/pequod-engine)
